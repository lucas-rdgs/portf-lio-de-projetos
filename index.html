    Meu Portfólio

    Projeto de regressão para previsão de vendas de unidades de uma rede de farmácias

    Utilizando Python, foram implementados modelos de Machine Learning para realizar a previsão para 6 semanas no futuro para as lojas de uma rede de farmácias. Dentre os modelos de regressão testados, foi utilizado o regressor XGBoost que conferiu um erro percentual absoluto médio (MAPE) de 11,15% por loja.

    Foi desenvolvido ainda um bot no aplicativo de mensagens Telegram em que o usuário escolhe a unidade da rede a qual deseja receber a previsão de vendas.

    As ferramentas utilizadas foram:
  • Python, Pandas, Numpy, Matplotlib, Seaborn
  • Sklearn, Statistics
  • Git, GitHub, Jupyter Notebook, Pycharm, Heroku Cloud, Telegram

    Projeto de insights para aumentar o faturamento de uma empresa fictícia do ramo imobiliário

    Este projeto provê uma análise exploratória robusta de uma base de dados de imóveis disponíveis para compra na região de Seattle - EUA. Foram desenvolvidos dois dashboards com tabelas, mapas e gráficos detalhando as propriedades para auxiliar o público final nas tomadas de decisão.

    Caso todo o portfólio de imóveis seja comprado e revendido seguindo as premissas do projeto, o faturamento da empresa seria de 28%.

    As ferramentas utilizadas foram:
  • Python, Pandas, Numpy, Matplotlib, Seaborn, GeoPandas
  • Mapas interativos com Plotly e Folium
  • Anaconda, Jupyter Notebook, Pycharm
  • Git, GitHub, Heroku Cloud, Streamlit Python framework web
